# 🧠 AI Insurance Assistant Backend

An AI-powered backend system that parses insurance policy PDFs, structures them into meaningful data, and enables intelligent, context-aware question answering using LLMs.

---

## 🚀 Overview

This backend is the core engine of the **AI Insurance Assistant App**. It allows users to:

- 📄 Upload insurance policy PDFs
- 🤖 Extract structured data using AI
- 💾 Store processed policies in MongoDB
- 💬 Ask questions about policies
- ⚖️ Get AI-based verdicts (VALID / NOT VALID / DEPENDS)
